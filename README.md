Модель построена в программном комплекс BioUML [(Kolpakov et al., 2019](https://doi.org/10.1093/nar/gkz440), [2022)](https://doi.org/10.1093/nar/gkac286) в SBML-формате [(Keating et al., 2020)](https://doi.org/10.15252/msb.20199110 ) с использованием графической нотации SBGN [(Le Novère et al., 2009)](https://doi.org/10.1038/nbt.1558 ).

![Финальная модель](https://github.com/DBgentech2023sirius/Cancer_treatment_FT_Grant_2025/blob/main/Images/Final_model_picture.png?raw=true)

Модель описывает лечение колоректальной карциномы CT26 у мышей.

Модель состоит из двух компартментов: 
<ul>
    <li>Кровь (blood)</li>
    <li>Опухоль (tumor)</li>
</ul>

В этих компартментах находятся 5 типов клеток: 
<ul>
    <li>Опухолевые клетки (Tu)</li>
    <li>Зараженные вирусом везикулярного стоматита опухолевые клетки (Ti)</li>
    <li>M1-макрофаги (M1)</li>
    <li>M2-макрофаги (M2)</li>
    <li>CD8+ T-клетки (Teff)</li>
</ul>

Модель включает 4 типа иммунотерапевтических препаратов:
<ul>
    <li>Анти-PD1 антитела (Ab_PD1)</li>
    <li>Вирус везикулярного стоматита (VSV)</li>
    <li>мРНК mGMCSF (mRNA_mGMCSF)</li>
    <li>мРНК mIL12 (mRNA_mIL12)</li>
</ul>

В модели описывались следующие режимы лечения:
<ul>
    <li>Монотерапии:</li>
    <ul>
        <li>Трехкратное внутриопухолевое введение липидных наночастиц с мРНК mIL12 (2.5 мкг)</li>
        <li>Трехкратное внутриопухолевое введение липидных наночастиц с мРНК mGMCSF (2.5 мкг)</li>
        <li>Трехкратное внутриопухолевое введение VSV (7.0E5 PFU)</li>
        <li>Трехкратное внутривенное введение анти-PD1 антител (200 мкг)</li>
    </ul>
    <li>Комбинированные режимы лечения:</li>
    <ul>
        <li>Однократное внутриопухолевое введение смеси липидных наночастиц с мРНК mIL12 (2.5 мкг) и мРНК mGMCSF (2.5 мкг)</li>
        <li>Трехкратное внутриопухолевое введение смеси липидных наночастиц с мРНК mIL12 (2.5 мкг) и мРНК mGMCSF (2.5 мкг)</li>
        <li>Шестикратное внутриопухолевое введение смеси липидных наночастиц с мРНК mIL12 (2.5 мкг) и мРНК mGMCSF (2.5 мкг)</li>
        <li>Трехкратное внутриопухолевое введение смеси липидных наночастиц с мРНК mIL12 (2.5 мкг) и мРНК mGMCSF (2.5 мкг) и VSV (7.0E5 PFU)</li>
        <li>Трехкратное внутриопухолевое введение смеси липидных наночастиц с мРНК mIL12 (2.5 мкг) и мРНК mGMCSF (2.5 мкг), VSV (7.0E5 PFU) и трехкратное внутривенное введение анти-PD1 антител (200 мкг)</li>
    </ul>
</ul>
